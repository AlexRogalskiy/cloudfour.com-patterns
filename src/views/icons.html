Icons

This toolkit includes a custom library of icons to accompany content where appropriate. These icons will inherit the parent element's text color where specified by the icon designer.

How to use

Reference the icons.svg file with the hash of the icon you wish to use (e.g. #arrow) in a standard SVG element:

<svg class="Icon">
  <use xlink:href="assets/toolkit/images/icons.svg#foobar" />
</svg>

The .Icon component class is included for consistent styles. This can be combined with modifiers to adjust the icon's size relative to the current font-size:

.Icon
.Icon.Icon--md
.Icon.Icon--lg
.Icon.Icon--xl
